# Project 2: Exploratory Data Analysis (EDA)

## Objective
Analyze the dataset to identify patterns, trends, distributions, and outliers.

## Tasks Performed
- Calculated Mean, Median and Count
- Created product-wise sales analysis
- Analysed payment methods
- Studied monthly sales trends
- Performed outlier analysis
- Generated business insights

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts

## Dataset Details
Rows: 1200  
Columns: 14

## Key Findings
- Identified sales patterns and trends
- Analysed customer payment preferences
- Found top-performing products
- Detected unusual transactions

## Conclusion
EDA helped convert raw data into meaningful insights through analysis and visualization.
